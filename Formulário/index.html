1. O que é VisualG?

O VisualG é uma ferramenta utilizada para o aprendizado de lógica de programação.

Ele permite a construção de algoritmos utilizando pseudocódigo.

Mesmo que limitado, ainda se pode elaborar programas incríveis se souber o que está fazendo.

Portanto, com o básico da lógica, dedicação e criatividade, tudo (ou quase) é possível.

Principais Objetivos

  • Aprender lógica de programação
  • Desenvolver algoritmos
  • Compreender estruturas básicas

2. Estrutura Básica de um Algoritmo

Início e Fim

Todo algoritmo possui início e fim bem definidos.

Pode haver sub-algoritmos dentro de outro algoritmo, possuindo também início e fim.

Declaração de Variáveis

As variáveis são utilizadas para armazenar valores.

  • INTEIRO - Para valores positivos e negativos sem vírgulas
  • REAL - Para valores fracionais (com vírgula), positivos e negativos
  • CARACTER - Para guardar texto, este não podendo efetuar operações matemáticas mesmo se número
  • LOGICO - Para apenas dois valores, positivo ou negativo

Observação

É importante manter a organização e identação do código.

3. Exemplos

Funções básicas:

Comandos Função
Algoritmo Define o nome do algoritmo
Var Declara variáveis
Inicio Marca o início da execução
Fimalgoritmo Marca o fim da execução
Escreva Demarca onde e o que será imprimido na tela
Escreval Função escreva somada a um espaçamento de pular linha
Leia Guarda o valor digitado pelo usuário na variável selecionada

Operadores Lógicos (usados em estruturas):

Operador Função
> Maior que
< Menor que
>= Maior ou igual
<= Menor ou igual
= Igual
<> Diferente

Estruturas de Decisão e Repetição:

Comando Função
Se () Então ... Fimse Executa uma ação caso a condição seja verdadeira
Repita ... Ate () Repete uma ação enquanto a condição for verdadeira (executa, depois valida)
Enquanto () Faça ... Fimenquanto Repete uma ação enquanto a condição for verdadeira (valida, depois executa)

Observação

Pode-se colocar estruturas dentro de estruturas.
